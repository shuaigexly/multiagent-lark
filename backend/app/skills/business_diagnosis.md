# 经营诊断技能

## 经营健康三角诊断

```
收入增长 ←→ 成本控制 ←→ 利润质量
    ↕              ↕              ↕
  规模           效率           可持续性
```

三角诊断：
- 收入增长 + 成本上升 + 利润持平：**增长模式有效，规模效应待验证**
- 收入平稳 + 成本下降 + 利润提升：**效率优化阶段，需警惕成本削减是否影响增长**
- 收入下降 + 成本刚性 + 利润恶化：**危险信号，需立即诊断收入下降根因**
- 三项同时下降：**系统性危机，需外部因素或内部重大变化解释**

## KPI 异常判断规则
| 变化类型 | 阈值 | 必须解释 | 是否紧急 |
|---------|------|---------|---------|
| 单期环比下降 | >15% | ✅ | 视持续性 |
| 连续3期下降 | 任意幅度 | ✅ | ✅ |
| 与预算偏差 | >20% | ✅ | ✅ |
| 与同期偏差 | >30% | ✅ | 视原因 |

## 季节性调整方法
分析数据前，必须先判断是否存在季节性因素：
1. 与去年同期对比（YoY）：排除季节性影响
2. 与移动平均线对比：判断趋势方向
3. 节假日效应：春节/国庆前后 ±2周数据异常属正常
4. 行业周期：B2B产品季末冲量是正常现象，不需要过度解读

## 经营问题根因链
收入问题：
- 新客减少 → 获客渠道/品牌问题
- 老客流失 → 产品/服务质量问题  
- 客单价下降 → 产品价值感/竞争定价问题
- 交易频次降低 → 场景培育不足/替代品竞争

成本问题：
- 人力成本飙升 → 规模扩张期正常，关注人效
- 获客成本上升 → 渠道竞争加剧，需多元化
- 运营成本比例上升 → 系统效率问题或规模不经济

## 经营报告必须回答的5个问题
1. 本期最好的信号是什么？（避免只谈问题）
2. 最值得关注的风险是什么？
3. 与计划/目标的偏差，主要由什么驱动？
4. 当前判断的数据置信度如何？（缺失哪些数据）
5. 未来30天最需要关注的1个先行指标是什么？
