# 飞书文档输出规范

## 文档结构模板

```
# 文档标题（简洁，包含时间/主题）

> 📌 **核心结论**：一句话概括最重要的判断（≤30字）

---

## 一、[章节名称]
**结论**：[本章核心判断]

[支撑内容，3-5条，用数据说话]

| 指标 | 本期 | 上期 | 变化 |
|------|------|------|------|
| xx   | xx   | xx   | ↑/↓  |

## 二、[章节名称]
...

---
## 行动建议
- [ ] [负责人] [截止日期] [具体行动]
- [ ] ...

---
*生成时间：{日期} | 数据来源：{来源}*
```

## 格式规则

### 必须使用的元素
- **引用块** `>` ：高亮核心结论，每文档至少1个
- **任务列表** `- [ ]`：所有行动项必须用此格式，便于飞书任务转换
- **表格**：超过3行的对比数据必须用表格
- **分割线** `---`：章节间必须有分割线

### 禁止的格式
- 连续超过5行的纯文字段落（必须拆解或用列表）
- 无标题的内容块（每个 ## 章节必须有内容）
- 嵌套超过2层的列表

### 数值呈现规范
- 百分比变化：用 ↑12% 或 ↓8% 箭头标注方向
- 金额：万元单位，保留1位小数
- 时间：精确到周（如：W15, 4月第3周）

## 飞书特有功能提示
在报告末尾加上：
```
💡 **飞书操作提示**：
- 将"行动建议"中的任务一键转为飞书任务（点击☐图标）
- 将此文档转发到相关群：@相关人员 + 分享文档链接
```
